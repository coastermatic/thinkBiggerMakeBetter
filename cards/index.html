Example
Here are 2 sets of priorities for the same project.

A goal setting feature for a social healthcare app.

  • Sharing the new goal with friends.
  • Giving the user power to set the right goal.
  • The new system is effective at helping the user stick to those goals.
  • The new code needs to be autonomous from the existing base.
  • Speed is an important factor in load times, since the new goal feature could be processing intensive.
  • Accessing the goal feature should be flexible and available across devices.
